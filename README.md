# Sales Performance Analytics Dashboard using Power BI

## Project Overview
This project focuses on analyzing sales and profit data using Power BI. 
An interactive dashboard was created to understand sales performance, regional trends, product performance, and category-wise analysis.

## Tools Used
- Power BI
- CSV Dataset

## Dashboard Features
- Total Sales Analysis
- Total Profit Analysis
- Sales by Region
- Profit by Region
- Product-wise Sales Analysis
- Category-wise Sales Analysis
- Sales Trend Analysis
- Interactive Filters (Region & Category)
- Top 5 Products by Sales
